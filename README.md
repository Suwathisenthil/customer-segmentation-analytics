# Customer Segmentation Analysis using Machine Learning

## Overview

Customer Segmentation is a data analytics project that groups customers based on purchasing behavior, demographics, and spending patterns. The objective is to help businesses understand their customers better and create targeted marketing strategies that improve customer retention and maximize revenue.

This project implements an end-to-end machine learning pipeline, starting from data preprocessing to customer segmentation, business insights, interactive dashboards, and executive reporting.

---

## Objectives

- Perform customer data cleaning and validation.
- Conduct Exploratory Data Analysis (EDA).
- Engineer meaningful customer features.
- Compare multiple clustering algorithms.
- Identify customer segments.
- Generate business KPIs.
- Produce interactive dashboards.
- Generate executive summaries and recommendations.
